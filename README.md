## ✨ 核心特色

- **中英文准确对应**：不信你对照一下。
- **高级英语加持**：精心调教的高级英语。
- **玄学浅析引擎**：将你的名字与风水、八字、星座紧密关联
- **MBTI人格**：把你的人格特质细致解读
- **像素风UI**：中国风＋像素风的UI设计，同时还隐藏了一只会动的魔法小猫，复古游戏风格界面，让起名这件事回到8-bit年代

## 🚀 使用方法

1. 输入你的英文名（必填）
2. 添加MBTI人格类型（可选）
3. 选择性别（可选）
4. 点击"生成中文名"按钮
5. 欣赏三个独属于你或者你的朋友的中文名方案
6. 保存你最喜欢的名片分享给朋友
